## 项目运行

#### 注意：由于涉及大量的 ES6/7 等新属性，nodejs 建议高版本 我使用的
>  node -v v8.1.2
> npm -v 5.0.4
> yarn -v 0.27.5
> 由于新版本的npm不太稳定，就用了yarn做下载工具，用起来还不错
```

git clone https://github.com/wuchaoya/react-web.git  

cd react-web

npm install 或者 yarn

npm start 或者 yarn start

npm run build 或者 yarn run build

```
